Odyn

Reproducible vendoring for Odin. Not a package manager.

Install (it's free)Get Started
Read the docs

Pin Once, Sync Anywhere

Add a dependency with odyn get and its commit hash is written to Odyn.lock. Run odyn sync on any machine to reproduce the exact same state.

No Registry, No Surprises

Odyn has no central registry and no transitive dependency resolution. Every dependency in your project is one you explicitly added.

Plain Git Under the Hood

Odyn clones real Git repos into odyn_deps/. No custom archive format, no proprietary index. Browse, audit, or replace any dependency with standard Git tools.
